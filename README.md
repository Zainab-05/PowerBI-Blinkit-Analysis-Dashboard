# PowerBI-Blinkit-Analysis-Dashboard

##  Overview

This project presents an interactive **Power BI dashboard** built to analyze Blinkit’s retail sales performance. The dashboard provides insights into sales trends, outlet performance, product categories, and customer ratings to support data-driven decision-making.

---

## Objectives

* Analyze overall sales performance and key KPIs
* Understand outlet-wise and location-based performance
* Identify high-performing product categories
* Enable dynamic filtering for deeper business insights

---

## 📊Key Metrics (KPIs)

*  **Total Sales:** $1.20M+
*  **Average Sales:** $141
*  **Average Rating:** 3.9
*  **Number of Items:** 8,523

### 🔹 Sales Analysis

* Time-series visualization of **Outlet Establishment vs Sales**
* Identifies trends and peak performance periods

### 🔹 Outlet Performance

* Tier-wise analysis showing **Tier 3 outlets as top contributors (~472K)**
* Comparison across outlet size and location

### 🔹 Product Insights

* Sales distribution across **item types and fat content**
* Category-level performance comparison

### 🔹 Customer Insights

* Analysis of **average ratings (~3.9–4)** across products
* Helps assess product quality perception

### 🔹 Interactive Filtering

* Slicers for:

  * Outlet Location Type
  * Outlet Size
  * Item Type
* Enables **dynamic drill-down and customized analysis**

##  Tools & Technologies

##  Key Insights

* Tier 3 outlets contribute the highest sales (~472K), indicating strong performance in those locations
* Sales distribution varies significantly across outlet types and sizes
* Product ratings are relatively consistent (~4), indicating stable customer satisfaction
* Certain product categories contribute more significantly to overall revenue

---

##  Business Impact

This dashboard enables stakeholders to:

* Identify high-performing outlets and optimize expansion strategies
* Analyze product-level performance for inventory decisions
* Monitor sales trends and customer satisfaction metrics
* Make **data-driven retail and operational decisions**
